1. Catálogo de productos (Funcional, Alta): El sistema debe permitir mostrar productos organizados por categorías (cuadernos, bolígrafos, material escolar, oficina, etc.)[cite: 2].
2. Búsqueda de productos (Funcional, Alta): Los usuarios deben poder buscar productos por nombre, categoría, marca o código[cite: 2].
3. Carrito de compras (Funcional, Alta): Los usuarios deben poder agregar, modificar cantidades y eliminar productos del carrito[cite: 2].
4. Gestión de usuarios (Funcional, Alta): El sistema debe permitir registro, inicio de sesión y recuperación de contraseña[cite: 2].
5. Proceso de checkout (Funcional, Alta): El sistema debe guiar al usuario a través del proceso de compra con datos de envío y facturación[cite: 2].
6. Gestión de pedidos (Funcional, Alta): El sistema debe registrar y mostrar el estado de los pedidos (pendiente, procesando, enviado, entregado)[cite: 2].
7. Métodos de pago (Funcional, Alta): Integración con pasarelas de pago (tarjetas, transferencias, pago contra entrega)[cite: 2].
8. Inventario (Funcional, Alta): Control de stock en tiempo real para evitar ventas de productos agotados[cite: 2].
9. Panel administrativo (Funcional, Alta): Interfaz para gestionar productos, pedidos, usuarios e inventario[cite: 2].
10. Perfil de usuario (Funcional, Media): Los clientes deben poder ver y editar su información personal y direcciones guardadas[cite: 2].
11. Historial de compras (Funcional, Media): Los usuarios deben poder consultar sus pedidos anteriores[cite: 2].
12. Sistema de descuentos (Funcional, Media): Aplicación de cupones y códigos promocionales[cite: 2].
13. Notificaciones por email (Funcional, Media): Confirmación de pedido, cambios de estado, recuperación de contraseña[cite: 2].
14. Calculadora de envío (Funcional, Media): Estimación de costos según destino y peso[cite: 2].
15. Wishlist/Lista de deseos (Funcional, Media): Los usuarios pueden guardar productos para comprar después[cite: 2].

19. Sistema de reseñas (Funcional, Media): Los clientes pueden calificar y comentar productos[cite: 3].

30. Responsive design (No Funcional, Alta): Adaptación completa a dispositivos móviles, tablets y desktop[cite: 3].
